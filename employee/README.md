# 👥 HR Analytics Dashboard

## 📌 Project Overview

This project is an interactive **Power BI HR Analytics Dashboard** designed to help HR managers monitor workforce performance, employee satisfaction, resignation trends, training effectiveness, and salary distribution. The dashboard provides actionable insights to support strategic workforce planning and improve employee engagement.

---

## 🎯 Business Objective

The dashboard enables HR teams to:

- Monitor employee satisfaction and engagement.
- Analyze employee performance across departments.
- Track resignation rates and workforce retention.
- Evaluate training effectiveness.
- Compare salary distribution by job title.
- Support data-driven HR decisions.

- Dynamic slicers
